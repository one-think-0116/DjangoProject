Meet Django

Django is a high-level Python Web framework that encourages rapid development and clean, pragmatic design. Developed by a fast-moving online-news operation, Django was designed to handle two challenges: the intensive deadlines of a newsroom and the stringent requirements of the experienced Web developers who wrote it.

Ridiculously fast.

Django was designed to help developers take web applications from concept to completion as quickly as possible.

Reassuringly secure.

Django takes security seriously and helps web developers avoid many of the common security mistakes.

Exceedingly scalable.

Some of the busiest websites on the web leverage Django's ability to quickly and flexibly scale.
